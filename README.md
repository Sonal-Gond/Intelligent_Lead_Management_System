# Intelligent_Lead_Management_System

This scenario automates lead enrichment and management by integrating email verification, person and company enrichment, and lead source tracking.

**Requirements:**
Make.com Account
Create API of Hunter io
connect the airtable,google sheet, typeform, Linked Lead Form modules with your account.

**Importing the Scenario:**
Download Json file.
Open your Make.com dashboard.
Click “Create a new scenario”.
Click “Import scenario” (usually available at the top-right menu).
Upload the scenario JSON file provided.
Wait for the import to complete — all modules and connections will appear.
